# sfm-bathy-mapper
Tool for extracting bathymetry data from structure from motion (sfm) photogrammetry. 

## 1. Install UV
Firstly, install UV for create python environment, then check uv version and python version
```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
uv --version
python --version
```
